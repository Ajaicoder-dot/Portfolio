# Resume Portfolio

This is a full-stack resume portfolio application using Express, Vite, and React.
It has been configured to run with in-memory storage, so **no database setup is required**.

## Prerequisites

- Node.js (v20 or newer recommended)

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```

## Running the Application

1. Start the development server:
   ```bash
   npm run dev
   ```
   The application will run on [http://localhost:5000](http://localhost:5000).

## Features

- **In-Memory Storage**: Data is stored in memory and reset when the server restarts.
- **Pre-populated Data**: The application comes seeded with sample projects, skills, experience, and education data.
